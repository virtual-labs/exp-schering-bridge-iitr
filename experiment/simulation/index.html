STEP 1 : Make all the Correct Connections.

1.    M1 of MCB - M3 of ScheringKit

2.    M2 of MCB - M4 of ScheringKit

3.    B1 of Set Amplitude - B3 of Sine Wave

4.    B2 of Set Amplitude - B4 of Sine Wave

5.    D1 of Null Detector - D3 of Digital Null Detector

6.    D2 of Null Detector - D4 of Digital Null Detector

7.    X1 - S1 of Capacitor

8.    X2 - S2 of Capacitor

9.    X3 - X4

STEP 2 : Click on "CHECK" Button.    

STEP 3 : Click on the MCB to Turn ON the circuit.

STEP 4 : Click on Set Amplitude to set amplitude by clicking on knob.

STEP 5 : Select the values from various dropdowns to set minimum value of null detector

STEP 6 : Click "ADD" button to add the values to the Observation Table.

STEP 7 : If combination is wrong then select another combination.

STEP 8 : Add different values to the Table by selecting different value of multipliers.

STEP 9 : Click on "PRINT" button to print the page.
